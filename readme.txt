BusaTools integrates the BusaTools bst.js script, enabling website analytics, user session recording, heatmaps, live chat, feedback forms, and logs.

== Description ==

BusaTools is designed to seamlessly integrate a suite of powerful website optimization tools into your WordPress site. By inserting the BusaTools bst.js script into your site, you unlock a comprehensive set of features aimed at understanding and enhancing user experience.

**Features include:**
- **Website Analytics:** Gain insights into your website traffic and user behavior.
- **User Session Recording:** Watch real user sessions to understand user interactions.
- **Heatmaps:** Visualize where users click, scroll, and focus on your site.
- **Live Chat:** Engage with your website visitors in real-time to boost conversions.
- **Feedback Forms:** Collect valuable user feedback to improve your website.
- **Error and Regular Logging:** Keep track of any issues and user activities on your site.

With BusaTools, optimizing your website for better user engagement and conversion has never been easier.

== Installation ==

1. Upload the `busatools` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The BusaTools bst.js script will automatically be added to your site, enabling all functionalities.

== Frequently Asked Questions ==

= Do I need to configure any settings? =

No, there are no settings to configure. Once activated, BusaTools automatically adds the bst.js script to your site.

= Is BusaTools free to use? =

The core functionalities of BusaTools are free. Some advanced features may require a subscription. Please visit [BusaTools' website](https://busatools.com/) for more information.
